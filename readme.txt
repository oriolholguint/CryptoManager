-- Crypto Manager --

Descripción del programa:
-> Los usuarios se registran.
-> Pueden seleccionar diferentes tipos de cryptomonedas
-> Insertar la cantidad que poseen y el dinero FIAT (euro, dolar...) con el que las compraron.
-> Aparece el valor total de inversión en FIAT y BTC
-> Resumen del historial de compra/venta.


Tecnologías utilizadas:
-> Java con JSwing
-> Base de datos con SQL
